      Chrome 扩展

      常用 Chrome 扩展工具，链接多为 Chrome Web Store 项目页面。

      # 浏览器

      • 扩展管理器open in new window：快速管理扩展，对扩展分组启用。

      • Bookmarks clean upopen in new window：清理重复书签、空文件夹和失效链接，也能合并重复文件夹。

      • Better Historyopen in new window：按日期、小时罗列历史记录，搜索更便捷。

      • Cookie Editoropen in new window：管理、修改、导出 Cookie。

      # 标签页

      • iTabopen in new window：自定义你的新标签页。

      • zvTabsopen in new window：一键关闭左侧标签。

      • GoTo Tabopen in new window：关键词搜索，快速定位标签页。

      • Tab Wrangleropen in new window：自动关闭不活动的标签页，并允许您轻松还原找回它们。

      # 链接批量

      • Bulk URL Opener Extensionopen in new window：批量打开链接，也可批量获取当前浏览器内所有页面网址链接。

      • Link Grabberopen in new window：批量提取、筛选、复制网页里各种链接。

      • Linkclumpopen in new window：按住 Z 键后，长按鼠标左键进行区域框选，区域内的链接可以进行批量打开、复制、书签，操作中会同步显示区域内的链接数。

      • TabCopyopen in new window：快速复制标签页链接。

      # 图片视频

      • ImageAssistantopen in new window：用于嗅探、分析网页图片并提供批量下载。

      • Fatkun 图片批量下载open in new window：找出当前页面的所有图片，提供按分辨率、链接等筛选图片。

      • FireShotopen in new window：捕捉网页截图。

      • Screenityopen in new window：屏幕录像工具，可对屏幕进行捕获，注释，编辑、标注等。

      • Picture-in-Picture Extensionopen in new window：视频画中画。

      • Video Speed Controlleropen in new window：使用快捷方式加快，减慢，推进和回放 HTML5 视频。

      # 监视爬虫

      • Auto Refresh Plus | Page Monitoropen in new window：定时刷新页面；页面监视器，找到或丢失指定文本时，弹出提示。

      • Distill Web Monitoropen in new window：监控网页或源以获取变更，可邮件、手机提示。

      • Check 酱open in new window：网页内容监控工具，可以监测网页内容变化，并发送异动到微信/飞书。Check 酱可以部署在 Docker 或浏览器插件。

      • Web Scraperopen in new window：从网页中提取数据的爬虫。

      • 网页自动化：UI.Vision PRAopen in new window, iMacrosopen in new window, Automaopen in new window。

      # 网页优化

      • Tampermonkeyopen in new window：油猴脚本插件，可修改网页布局、增减内容、自动化操作，常用脚本库为 Greasy Forkopen in new window。

      • Fix Contrastopen in new window：自动校正网页对比度，让网页内容更易阅读和分辨。

      • FasterChromeopen in new window：浏览器预加载链接，提升网络流畅度。

      • SuperCopyopen in new window：一键破解禁止右键、破解禁止选择、破解禁止复制、破解禁止粘贴，启用复制，启用右键，启用选择，启用粘贴。

      • AutoPagerizeopen in new window：自动识别 next 或下一页，将网页合并为同一页。2014 年更新，很多规则已失效。

      • pakkuopen in new window：合并 B 站视频中绝大多数刷屏弹幕。

      • Enhanced GitHubopen in new window：提升 GitHub 易用度。

      • Sourcegraphopen in new window：优化 GitHub 代码，支持鼠标悬停、代码搜索、查看引用等。

      • 沉浸式翻译open in new window：免费的双语对照网页翻译，彩云小译open in new window 的替代品。

      # 实用工具

      • Tangoopen in new window：将页面操作转化为逐步指导的流程指南。每一步都自动植入截图，并可以通过链接、HTML、Markdown 来分享。

      • SingleFileopen in new window：将完整网页保存到单个文件中，可对文件名、HTML 内容、样式、图片、字体等进行调整。

      • Vimium Copen in new window：全键盘操作浏览器，建议启用 chrome://flags/#extensions-on-chrome-urls。

      • SelectorGadgetopen in new window：轻松获取网页元素的 CSS Path 或 XPath。

      • Copy as Markdownopen in new window：将网页文字转为 Markdown 格式文本。

      • Get Faviconopen in new window：快速获取当前网站的图标。

      • Similar Sitesopen in new window：发现与当前浏览的网站相似的其他网站。

      • Wappalyzeropen in new window：查看竞争对手网站使用了哪些技术、工具和第三方服务。

      • Link to Text Fragmentopen in new window：在 Chrome 浏览器中分享网页并突出显示所选文本。
